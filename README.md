> **Ordinary days deserve beautiful things.**

Kikuyu Room is an independent lifestyle brand website built around **memory, imagination, and everyday life**.

Rather than creating a simple shopping demo, this project combines **brand identity, UI/UX design, frontend development, and AI-assisted development** into one complete portfolio project.

# ✨ Project Overview / 项目简介

Kikuyu Room is built around one simple idea:

**A Room is not simply a place. It is a vessel for time.**

Room 不只是一个房间，它是一个承载时间的容器。

Every product belongs to one of the **Four Tiny Worlds**, representing different ways of living rather than traditional product categories.

每件商品都属于 **Four Tiny Worlds（四个微小世界）** 中的一个世界，它们代表的是一种生活方式，而不是传统意义上的商品分类。

This project simulates the complete workflow of building a modern independent brand—from **brand strategy → UI design → frontend development → deployment**.

整个项目模拟了一个独立品牌从 **品牌策划 → UI设计 → 前端开发 → 网站部署** 的完整流程。

# 🌸 Project Highlights / 项目亮点

Unlike a traditional e-commerce demo, Kikuyu Room focuses on creating a complete brand experience.

不同于普通商城练习项目，Kikuyu Room 更注重打造完整的品牌体验。

### This project includes / 本项目包含
